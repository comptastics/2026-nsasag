\section{Some minor notes on discussion}

\paragraph{Responsible Parties:} Lucas (primary), Tatsu

\begin{enumerate}[1.]
\item They know all the class labels, but the interesting thing is to find
  where the $Y$ class labels are particularly high.
  %
  Motivating example: $X$ are salmon, $Y$ are orcas, and would like to know
  if orcas congregate even when there are not salmon.
\item The coordinates are spatio-temporal: $(x, y, t)$ pairs, where $(x, y)$
  are location and $t$ istie
\item Lucas thought maybe one could obtain some guarantees that say ``this
  region is good'' using his work on subgraouup ID in clinical trials.
\item they do not appear to have side information to form nice
  feature vectors -- just the location information -- so it is not
  100\% clear if a strategy based purely on classification
  and then inverting a classification rule to find densities would
  be good.
  %
  But it would be nice to have a few notes on it regardless.
\end{enumerate}
